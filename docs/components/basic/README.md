## Icon 图标
提供了一套常用的图标集合。

<h3>使用方法</h3>

直接通过设置类名为 `o-icon-iconName` 来使用即可。例如：
<ClientOnly>
  <icon-demo></icon-demo>
</ClientOnly>

```html
<i class="o-icon-home"></i>
<i class="o-icon-delete"></i>
<i class="o-icon-view"></i>
<i class="o-icon-search"></i>
```
